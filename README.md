EVE Year in Review
==================

Showcase of new character stats for EVE Fanfest 2015

Dev Prerequisites
-----------------

* node + npm (built with 0.10.35)

Get Started
-----------

    npm install
    alias gulp='node_modules/.bin/gulp'
    gulp iconFont
    gulp

If this all works, gulp should assemble all the pieces and open up your browser in
development mode.

Production Build
----------------

    gulp production
